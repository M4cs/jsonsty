JSONsty API

We offer an API to make integrating our service as seamless as possible for you. Below you will find documentation on said API.

Getting Started

Sign Up for the JSONsty service and remember your username and password.

BASE URL: https://json.psty.io/api_v1/

Authorization

Login

Parameter Description Type
email E-mail you used to sign-up for JSONsty String
password Your account password used to sign in to JSONsty String

This endpoint will return a status message and if successful, an access token.

URL: POST https://json.psty.io/api_v1/login

Example Request:

>> import requests
>> res = requests.get('https://json.psty.io/api_v1/login', data={'email': 'test@example.com', 'password': 'example123!')
>> print(res.json)
{ "message": "Logged In!", "access_token": "<ACCESS_TOKEN>" }

All requests require including the access_token in the header passed as access_token.

Creating Stores

Parameter Description Type
data JSON Data to Post JSON

URL: POST https://json.psty.io/api_v1/create

Example Request:

>> import requests, json
>> headers = { 'access_token': '<ACCESS-TOKEN>', 'Content-Type': 'application/json' }
>> data = {'test_data': { 'test_array': ['test_val1', 'test_val2'] }
>> res = requests.post('https://json.psty.io/api_v1/create', json=data)
>> print(res.json)
{ "message": "Success", "name": name }

Getting Store

URL: GET https://json.psty.io/api_v1/stores/<STORE_NAME>

Example Request:

>> import requests
>> headers = { 'access_token': '<ACCESS-TOKEN>' }
>> res = requests.get('https://json.psty.io/api_v1/stores/<STORE_NAME>')
>> print(res.json)
{'test_data': { 'test_array': ['test_val1', 'test_val2'] }

Editing Store

URL: PUT https://json.psty.io/api_v1/stores/<STORE_NAME>

Example Request:

>> import requests
>> headers = { 'access_token': '<ACCESS-TOKEN>', 'Content-Type': 'application/json' }
>> res = requests.get('https://json.psty.io/api_v1/stores/<STORE_NAME>')
>> new_data = res.json.copy()
>> new_data.update({'new_key': 'new_value')
>> res = requests.put('https://json.psty.io/api_v1/stores/<STORE_NAME>', data=new_data)
>> print(res.json)
{'test_data': { 'test_array': ['test_val1', 'test_val2'] }

Deleting Stores

URL: DELETE https://json.psty.io/api_v1/stores/<STORE_NAME>

Example Request:

>> import requests
>> headers = { 'access_token': '<ACCESS-TOKEN>' }
>> res = requests.delete('https://json.psty.io/api_v1/stores/<STORE_NAME>')
>> print(res.json)
{ "message": "Success" }
